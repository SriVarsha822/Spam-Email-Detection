# 📧 Spam Email Detection using Machine Learning

## 📌 Overview

This project predicts whether an email is **Spam** or **Genuine** using Machine Learning.

## 🚀 Features

- Spam Detection
- Confidence Score
- Spam & Genuine Probability
- AI Reason
- Modern Flask UI

## 🛠 Technologies

- Python
- Flask
- Pandas
- Scikit-Learn
- HTML
- CSS

## ▶️ Run

```bash
python app.py
```

## 🤖 Algorithm

Multinomial Naive Bayes

## Accuracy

96.86%
