# 📧 Spam Email Detection Using Machine Learning

## 📌 Overview

This project is a Machine Learning web application that classifies emails as **Spam** or **Genuine** using the Multinomial Naive Bayes algorithm. It provides real-time predictions through a Flask-based web interface.

---

## 🚀 Features

- Detects Spam and Genuine Emails
- Confidence Score Display
- Spam & Genuine Probability Visualization
- Prediction Explanation
- Responsive Flask Web Interface

---

## 🛠️ Technologies Used

- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- HTML5
- CSS3
- Git & GitHub

---

## 🤖 Machine Learning Algorithm

- Multinomial Naive Bayes

**Model Accuracy:** 96.86%

---

## ▶️ How to Run

```bash
python app.py
```
